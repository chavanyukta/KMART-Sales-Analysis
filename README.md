# Kmart Sales Dashboard

## Overview
This project transforms raw Kmart sales data into actionable insights using Excel. It demonstrates data cleaning, analysis, and dashboard creation to provide a comprehensive view of business performance.

## Dataset
The dataset includes:
- Sales across states, suburbs, and categories  
- Online vs in-store sales  
- Manager and customer information  

## What I Did
1. **Data Cleaning & Organization**  
   - Structured raw data into multiple sheets for easy analysis  

2. **Analysis Using Excel**  
   - Calculated totals, averages, and trends using formulas and pivot tables  

3. **Data Visualization**  
   - Built charts for:  
     - Sales over time  
     - Top customers  
     - Top managers  
     - Category, state, and suburb breakdowns  

4. **Dashboard Creation**  
   - Created a user-friendly Excel dashboard for a quick performance snapshot  

## Key Insights
- **Total Sales:** $3,186,258  
- **Average Sales per Transaction:** $707  
- **Top Suburb:** Campbelltown  
- **Top Manager:** Rachael Long  
- **Top Category:** Home & Living ($308.91K)  
- **Store vs Online Sales:** Stores $10.9M (80%), Online $2.64M (20%)  
- **Top State:** NSW ($4.94M)
